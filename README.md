# arche-reader-controller

Arduino code for the xy reading process for the Arche-Scriptures project  

## Dingqi Eletronics Pin Mapping


|		Stepper 	|	ATMega328p 	 	|	Arduino  	 	|
|-------------------|-------------------|-------------------|
|		dir x 	 	|		PD5			|		D5			|
|		step X 		|		PD2			|		D2			|
|		en X		|		PB0			|		D8			|
|		dir Y		|		PD6			|		D6 			|
|		step Y 		|		PD3			|		D3 			|
|		en Y		|		PB0 		|		D8 			|
